# Graph Hierarchy Visualization (Vue 3)

## Description
Visualizes hierarchical graph data using Vue 3.
Users can click a node to view details and deselect it.

## Tech Stack
- Vue 3
- Vite
- CSS

## Features
- Tree hierarchy (top-down)
- Node selection & highlighting
- Sidebar with node details
- Deselect support

## Setup
```bash
clone the repo
npm install
npm run dev
